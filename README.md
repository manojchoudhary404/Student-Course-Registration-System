# 📚 Student Course Registration System

## 🚀 Project Overview

The **Student Course Registration System** is a console-based Java application that allows users to manage student records efficiently. It demonstrates core Java concepts such as Collections, Stream API, Lambda expressions, and Exception Handling.

---

## 🎯 Features

* ➕ Add Student details
* 📋 View all students
* 🎯 Filter students with marks greater than 70
* 📊 Group students by course
* 🔽 Sort students by marks (descending)
* ⚠️ Handle invalid inputs using custom exceptions

---

## 🛠️ Technologies Used

* Java (Core Java)
* Collections Framework (`ArrayList`)
* Stream API
* Lambda Expressions
* Exception Handling

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/Student-Course-Registration-System.git
   ```

2. Navigate to the project folder:

   ```
   cd Student-Course-Registration-System
   ```

3. Compile the Java files:

   ```
   javac com/student/model/*.java com/student/service/*.java com/student/app/*.java com/student/exception/*.java
   ```

4. Run the application:

   ```
   java com.student.app.StudentApp
   ```
